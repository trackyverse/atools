# Worker function to create filtering vector

Worker function to create filtering vector

## Usage

``` r
create_filter_vec(object, slt = c("ani", "dep", "det", "tag", "obs"), checks)
```

## Arguments

- object:

  an ATO

- slt:

  the ATO slot to filter

- checks:

  a list of column names and respective values to filter with

## Value

a logical vector
