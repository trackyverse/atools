# Clear all previously applied (non-hard) filters

Clear all previously applied (non-hard) filters

## Usage

``` r
filter_reset(object, s)
```

## Arguments

- object:

  an
  [`ATO`](https://trackyverse.github.io/ATO/reference/ATO_package.html)

- s:

  the slot to be reset. If not provided, all slots are reset.

## Value

the updated ATO
